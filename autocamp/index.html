Autocamp Zvůle

Kompletní webová aplikace pro správu autocampu — online rezervace, ceník, galerie a administrátorský panel.

🖥️ Serverová aplikace  —  vyžaduje Python/Flask server k provozu
← Zpět na portfolio

Funkce aplikace

📅
Online rezervace
Zákazníci si mohou rezervovat místo přímo z webu s výběrem termínu a typu umístění.
💰
Ceník
Přehledný ceník sezón, typů parcel a doplňkových služeb.
📸
Fotogalerie
Galerie fotek autocampu, přírody a vybavení.
🔐
Admin panel
Správa rezervací, ceníku a obsahu webu z administrátorského rozhraní.
📊
Přehled obsazenosti
Přehled aktuální obsazenosti campingu v kalendářním zobrazení.
📧
Potvrzení e-mailem
Automatické potvrzení rezervace zákazníkovi i majiteli.

Technologie

🐍 Python
🌶️ Flask
🗄️ SQLite
🌐 HTML / CSS / JS
📋 Jinja2
